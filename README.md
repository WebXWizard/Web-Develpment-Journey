## How to Run a Project

Each folder may contain a separate project. For projects that include a `package.json`, open that folder and run:

```bash
cd "folder-name"
npm install
npm run dev
```

For simple HTML, CSS, and JavaScript projects, open the `index.html` file directly in the browser.
